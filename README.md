## Installation

## Create venv
```
	python3 -m venv venv

```

## Activate venv
```

source venv/bin/activate

```

## Dependencies

```
pip3 install -r requirements.txt

```


## Run PayPark Project
```

python manage.py runserver

```

## Database Migrations
### Step 1 :: Generate Migrations File

```

python manage.py makemigrations

```

### Step 2 :: Migrate Migrations File

```

python manage.py migrate

```
